Presentation Mode Extension

This GNOME Shell extension adds a Presentation Mode toggle to the top bar, allowing you to easily disable screen dimming and locking during presentations.

Features

  • Toggle Presentation Mode on and off from the top bar
  • Prevents screen dimming and locking
  • Simple and easy to use

Installation

To install this extension, download it from the GNOME Extensions website or clone it from the repository and place it in the appropriate directory.

Usage

Once installed, you can enable the extension from GNOME Tweaks. Use the switch in the top bar to toggle Presentation Mode on and off.
